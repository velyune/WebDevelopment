# Имамов Д.Н — Запрос, оплата и получение копий нотариальных документов

## Описание

Разработать дизайн и HTML/CSS/JS страницы «Запрос, оплата и получение копий нотариальных документов» для ролей: Заявитель / Нотариус (по модели).

## Дизайн интерфейса

Спроектировать UI страницы «Запрос, оплата и получение копий нотариальных документов» с учетом сценариев использования и ролей.

## Функции из docs/Responsobility.md

- Форма запроса, прикрепление оснований, расчёт стоимости, оплата, выдача копий/ссылок, статус “в обработке/готово”.

## Ожидаемый результат

- Макеты (desktop + responsive).
- Верстка HTML/CSS.
- JS-интерактивы/стабы для основных действий.

## Доп. требования

- Доступ: Заявитель / Нотариус (по модели).
