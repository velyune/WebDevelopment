# Мирошник И.О. — Уведомления + история

## Описание

Разработать дизайн и HTML/CSS/JS страницы «Уведомления + история» для ролей: Заявитель / Нотариус / Админ.

## Дизайн интерфейса

Спроектировать UI страницы «Уведомления + история» с учетом сценариев использования и ролей.

## Функции из docs/Responsobility.md

- In-app уведомления, фильтры, прочитано/не прочитано, история событий, настройки каналов (email/push).

## Ожидаемый результат

- Макеты (desktop + responsive).
- Верстка HTML/CSS.
- JS-интерактивы/стабы для основных действий.

## Доп. требования

- Доступ: Заявитель / Нотариус / Админ.
