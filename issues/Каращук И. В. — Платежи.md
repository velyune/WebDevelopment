# Каращук И. В. — Платежи

## Описание

Разработать дизайн и HTML/CSS/JS страницы «Платежи» для ролей: Нотариус / Заявитель (если платит).

## Дизайн интерфейса

Спроектировать UI страницы «Платежи» с учетом сценариев использования и ролей.

## Функции из docs/Responsobility.md

- Выбор тарифа, ввод реквизитов, промокод (опц.), подтверждение оплаты, чеки/счета, история платежей.

## Ожидаемый результат

- Макеты (desktop + responsive).
- Верстка HTML/CSS.
- JS-интерактивы/стабы для основных действий.

## Доп. требования

- Доступ: Нотариус / Заявитель (если платит).
