# Angular 21 — пустой проект

Проект создан в папке `src` и готов к разработке.

## Требования

- Node.js 20+
- npm 10+

## Установка и запуск

```bash
cd src
npm install
npm start
```

Приложение откроется по адресу http://localhost:4200/

## Команды

- `npm start` — dev-сервер с hot reload
- `npm run build` — сборка для production в `dist/app`
- `npm test` — запуск unit-тестов (Vitest)

## Структура

- `src/app/` — корневой компонент и код приложения
- `src/assets/` — статические файлы
- `src/styles.css` — глобальные стили
