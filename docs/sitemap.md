# Карта сайта — Нотариальная оценка наследства

На основе [Responsobility.md](Responsobility.md) и [FR.md](FR.md).

---

## Области страницы (layout)

- **Слева:** боковое меню навигации
- **Сверху:** заголовок (header) с названием раздела и данными пользователя
- **Центр:** основная область контента
- **Снизу:** футер (footer) с ссылками и копирайтом

## Гость (неавторизованный)

| Путь | Страница | Функции |
|------|----------|---------|
| `/` | Главная (TestPage) | — |
| `/landing-page` | Landing page | Продажная страница |

## Переходы между ролями

- **Гость** → **Заявитель/Нотариус/Админ:** после входа через `/auth` редирект по роли
- **Заявитель/Нотариус** → **Гость:** выход из системы
- Общие страницы (FAQ, поддержка) доступны из меню каждой роли
